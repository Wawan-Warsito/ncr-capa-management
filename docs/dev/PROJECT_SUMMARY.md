# Project Summary

## Ringkasan
Project ini merupakan sistem manajemen NCR–CAPA berbasis web untuk mendukung pencatatan, approval, penugasan PIC, tindakan korektif/pencegahan, verifikasi efektivitas, dan pelaporan.

## Modul Utama
- NCR Management
- CAPA Management
- Dashboard (company/department/personal)
- Reports
- Notifications + Activity Log (audit trail)
- Import NCR (CSV/XLS/XLSX)

## Tech Stack
- Backend: Laravel 12 (PHP 8.2+)
- Frontend: React 19 + Vite 7
- Database: SQLite (local) / MySQL (opsional)
- Auth: Laravel Sanctum

## Cara Menjalankan (Local)
Lihat `docs/dev/QUICK_START.md`.

## Import data NCR dari CSV
Lihat `docs/dev/QUICK_START.md` atau gunakan menu NCR Management (Template + Import).
