# Frequently Asked Questions (FAQ)

## General

**Q: I forgot my password. How do I reset it?**
A: Currently, password resets are handled by the IT Administrator. Please contact IT Support at `it.support@company.com`.

**Q: Can I access the system from home?**
A: Yes, if you are connected to the company VPN. The system is web-based and accessible via browser.

**Q: What browsers are supported?**
A: We recommend using the latest versions of Google Chrome, Microsoft Edge, or Mozilla Firefox.

## NCR Process

**Q: I selected the wrong department. Can I change it?**
A: If the NCR is still in "Draft" status, you can edit it. If it has been submitted, please ask your Manager to "Reject" it so you can correct it.

**Q: Can I attach multiple photos?**
A: Yes, the system supports multiple file uploads (JPG, PNG, PDF). Each file should be less than 5MB.

**Q: Who approves my NCR?**
A: The Manager of the "Finder Department" you selected is responsible for the initial approval.

**Q: Why can't I close an NCR?**
A: Only the QC Manager has the permission to verify and close an NCR after all actions are completed.

## CAPA Process

**Q: When is a CAPA required?**
A: A CAPA is required for critical defects, recurring issues, or when the disposition is "Rework" or "Repair" that needs root cause analysis.

**Q: How do I update the progress of my assigned CAPA?**
A: Go to "My Tasks", open the CAPA, click "Update Progress", and enter the completion percentage and comments.

**Q: What if I cannot complete the action by the target date?**
A: Please add a comment in the CAPA explaining the delay and request an extension from the QA Manager.
